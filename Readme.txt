Quotation and cut list generation for your cut-to-size business.

== Description ==

This plugin uses the [SmartCut API](https://smartcut.dev/) to generate cut lists & quotations for your cut-to-size business.

## Features

- Linear, sheet or roll calculations
- Fast quotations
- Cut list diagrams
- Machining (e.g. holes, radius corners)
- Decimal & fractional units
- Edge banding for wood based sheet materials
- PDF & CSV of the cut requirements is attached to every order

## Pricing models

- Full stock
- Area of parts
- Cut length
- Full stock plus cut length
- Cut-to-size surcharge

== Installation ==

- Requires WooCommerce
- Install and activate the plugin.
- Translate with [Loco Translate](https://wordpress.org/plugins/loco-translate/) if you like.
- Create a cut-to-size product category.
- Head to Settings > SmartCut and enter your cut-to-size category.
- Head to Tools > SmartCut Templates to create template products.
- Sign up for an account at [SmartCut](https://smartcut.dev/) and add your domain.

== Privacy & data collection ==

- This plugin does not collect or use any personal data.
- To generate a calculations, the plugin sends data to the SmartCut API. No personal data is accessed, sent or stored.
- The privacy policy for the SmartCut API can be viewed [here](https://cutrevolution.com/privacy-policy.html).
